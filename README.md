# 🎓 Student Management System

A **Core Java** console-based application that helps manage student records efficiently. This project demonstrates fundamental Java programming concepts, including Object-Oriented Programming (OOP), collections, and menu-driven application development.

---

## 📌 Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- 🗑️ Delete Student
- 🚪 Exit Application

---

## 🛠️ Technologies Used

- Java
- Core Java
- Object-Oriented Programming (OOP)
- ArrayList
- Scanner Class

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/Brindhaj2701-Git/StudentManagementSystem.git
```

2. Open the project in your preferred Java IDE (Eclipse, IntelliJ IDEA, or VS Code).

3. Compile and run `Main.java`.

4. Use the menu to manage student records.
